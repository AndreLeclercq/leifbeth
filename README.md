### PC Principale (stream)
#### Hardware
* Intel NUCI3BEH
* Processeur i3 3 Ghz
* DDR4 16 Go
* SSD M2 240 Go
* HDD 1 To

#### Software
* Système d'exploitation : Linux Debian 11 (bullseye)
* Environnement de bureau : KDE Plasma
* Navigateur : Vivaldi
* IDE : Intellij IDEA Ultimate
* Stream : OBS Studio
